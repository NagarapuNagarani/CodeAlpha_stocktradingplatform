# Stock Trading Platform

A Java-based console application that simulates a basic stock trading environment. The application enables users to view available stocks, perform buy and sell operations, monitor their investment portfolio, and track portfolio performance. This project demonstrates the implementation of Object-Oriented Programming (OOP) concepts, Java Collections, and basic financial transaction management.

---

## Project Overview

The Stock Trading Platform provides a simple simulation of stock market trading. Users can browse available stocks, purchase shares, sell owned stocks, and view the total value of their portfolio. The project is designed to strengthen Java programming fundamentals and OOP principles.

---

## Features

- Display available market stocks
- Buy stocks with specified quantities
- Sell owned stocks
- View investment portfolio
- Calculate total portfolio value
- Console-based interactive interface
- Object-Oriented Programming (OOP) implementation
- Data management using ArrayList

## How to Run

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/StockTradingPlatform.git
```

### 2. Navigate to the Project Directory

```bash
cd StockTradingPlatform
```

### 3. Compile the Program

```bash
javac Stock.java Portfolio.java StockTradingPlatform.java
```

### 4. Run the Application

```bash
java StockTradingPlatform
```
